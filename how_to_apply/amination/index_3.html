令和３年度　京都府立山城高等学校第1回学校説明会について

 2021/09/13 
 

　既に御案内させていただいておりますように、令和３年度第1回学校説明会を９月25日（土）に予定しておりましたが、緊急事態宣言の期間延長に伴い、実施日を以下のように変更させていただきます。御予定いただいております皆様には御迷惑をおかけいたしますが、御理解いただきますようお願いいたします。



１　実施日　　令和３年10月２日(土)

２　場所　　　京都府立山城高等学校（変更ありません）

３　内容等　　

・感染の拡大状況を見ながら可能な限り予定通り実施いたします。

・来校時刻等の詳細は中学校より御配布いただいております「山城高校へ行こう！～新時代へつながる道～No.3」を御確認ください。

４　その他

　10月２日（土）の説明会実施及びその内容につきましては、感染状況をみながら実施日の一週間程前に再度判断をさせていただき、本校ホームページで案内させていただきます。

 

令和３年度第１回学校説明会のお知らせ

 2021/09/06 
 

第１回学校説明会のお知らせです。

実施される場所や内容につきましては、各中学校に配布いたしました「令和３年度第１回学校説明会のご案内」を御参照ください。


日程　令和３年９月2５日（土）

　　①９：00開始　上京区 / 中京区 / 下京区 / 左京区 / 北区

　　②11：00開始　伏見区 /乙訓・南山城・口丹通学圏 / その他の中学校

　　③13：50開始　東山区 / 山科区 / 南区 / 右京区 / 西京区

　　15：20終了

※校内の人数制限のため、90分での入れ換え制で実施します。

※通学する中学校の所在地によるグループ分けを参考に、可能な限り指定された時間に来校していただきますようお願いします。


内容

◇動画視聴　30分程度（令和４年度入学者選抜要項について）

◇個別相談　５分程度　※動画視聴後、希望される方のみ御案内します。

◇施設自由見学

※文理総合科の説明会は実施しません。


参加方法等

◇参加については、事前申し込みは必要ありません。

◇感染症拡大予防の観点から、一家庭につき２名様までの来校に御協力ください。

◇参加された方のお名前と座席を記録させていただきます。

◇当日は御来校前に検温等で体調を確認していただきますようお願いします。発熱がみられる場合は来校をお控えください。また、受付での検温と手指消毒に御協力ください。

◇学校説明会では、御自身でも『密』状態にならないよう周囲の方々と可能な限り距離をとって見学していただきますようお願いします。

◇本校は二足制ですので参加される方は上履きと下履き入れを御持参ください。

◇来校される際には公共交通機関を御利用ください。車・バイク・自転車での来校はお断りしております。

◇保護者の方で手話通訳を希望される場合は、事前に御連絡ください。


その他

◇学校説明や部活動紹介などは、山城高校広報YouTubeチャンネルにて公開しております。あらかじめ御視聴いただいた上でお越しください。文理総合科説明動画を９月13日（月）に追加します。YouTubeチャンネルのURL、QRコードは各中学校に送付いたしましたリーフレットに記載されております。

※京都府外、私立中学に通学されている方は、学校（075-463-8261）までお問い合わせください。

◇感染症拡大予防の観点から実施や内容等が中止、変更となる場合があります。その場合は学校HPで案内させていただきます。

◇説明会が予定通り実施できない場合等に備え、今年度は説明動画等をYouTubeで御覧いただけるようにしています。説明会の内容と同等の動画になっておりますので、十分に御活用ください。

◇学校説明会当日に活動する部活動については未定です。指定された時間内に活動している部活動に限り、御自由に見学いただけます。

◇すべての内容を見学いただくためには30分程度必要です。

◇終了時刻（15：20）にご協力ください。


 

9月12日（日）の部活動体験（見学）について

 2021/08/20 
 

8月20日（金）から９月12日（日）の期間で京都府に緊急事態宣言が発令されたことと、若年層において新型コロナウイルスの感染が急拡大している状況を鑑み、大変残念ではございますが、9月12日（日）の部活動体験（見学）は皆様の安心・安全のため中止の判断をさせていただきます。

申込フォームは本日をもって停止させていただきます。既に多くの中学3年生にお申し込みいただいている状況ではありますが、お申込いただいた皆様におかれましても参加していただくことはできません。部活動に関するお問い合わせ等がございましたら学校に御連絡ください。
